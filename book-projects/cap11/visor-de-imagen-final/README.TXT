Proyecto: VisorDeImagen-Final, version 3.1
Autores: David Barnes and Michael Kolling
Traducción: Maximiliano A. Eschoyez

Este proyecto es parte del material para el capítulo 11 del libro

   Programación Orientada a Objetos con Java
   Una Introducción Práctica usando BlueJ
   Tercera Edición
   David J. Barnes and Michael Kolling
   Pearson Educacion, 2007 

Este proyecto muestra la construcción de una IGU simple (incluyendo etiquetas
botones, gestores de disposición, manejo de eventos, menús, diálogos) como
también un poco de procesamiento de imágenes.

Para utilizar este proyecto, cree una instancia de la clase VisorDeImagen. Esto
automáticamente mostrará la IGU de la aplicación - no se requiere ninguna
interacción adicional con el objeto del banco de objetos.

ADVERTENCIA: 

A veces (desafortunadamente), ocurre que la ventana del visor de imágenes se
abre DETRÁS de la ventana de BlueJ. Esto puede resultar confuso, dado que
aparenta que nada ha ocurrido.
