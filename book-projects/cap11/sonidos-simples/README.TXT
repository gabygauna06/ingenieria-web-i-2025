Proyecto: sonidos-simples - Un reproductor simple de sonidos
Autores: David Barnes and Michael Kolling
Traducción: Maximiliano A. Eschoyez

Este proyecto es parte del material para el capítulo 11 del libro

   Programación Orientada a Objetos con Java
   Una Introducción Práctica usando BlueJ
   Tercera Edición
   David J. Barnes and Michael Kolling
   Pearson Educacion, 2007 

Este proyecto implementa un reproductor simple de sonidos, principalmente para
experimentar con la construcción de IGU.

Para utilizar este proyecto, cree una instancia de la clase ReproductorDeSonidoIGU.

Este reproductor puede reproducir archivos WAV, AIFF y AU con muestreo estándar
de 22.050 kHz o 11.025 kHz. (Esto significa que no se pueden reproducir todos
los archivos en estos formatos -- es importante la codificación.)
