Proyecto: libreta-de-direcciones-io
Autores: David Barnes and Michael Kolling
Traducción: Maximiliano A. Eschoyez

Este proyecto es parte del material para el capítulo 12 del libro

   Programación Orientada a Objetos con Java
   Una Introducción Práctica usando BlueJ
   Tercera Edición
   David J. Barnes and Michael Kolling
   Pearson Educacion, 2007 

Una implementación simple de una libreta de direcciones.
Esta versión incluye algo de verificación de la validez de los argumentos.

Cómo comenzar con este proyecto:
    Bien: Cree una objeto LibretaDeDirecciones y agregue detalles
    de contactos (como nombre, dirección y teléfono) a ella.
    O: Cree un objeto DemoLibretaDeDirecciones y llame al método
    getLibretaDeDirecciones.

Crea un LibretaManejadorDeArchivos para ilustrar algunas operaciones
básicas de entrada/salida en una LibretaDeDirecciones.

Instrucciones:
    + Los resultados de la búsqueda se guardarán en "resultados.txt"
      dentro del directorio del proyecto.
    + El archivo "ejemplo.txt", en el directorio del proyecto contiene
      una entrada de ejemplo que puede agregarse utilizando el método
      agregarEntradasDesdeArchivo de la LibretaManejadorDeArchivos.
    + Un objeto de libreta de direcciones completo puede grabarse, en
      formato binario, usando el método guardarEnArchivo. Puede ser leído
      con el método leerDeArchivo.
