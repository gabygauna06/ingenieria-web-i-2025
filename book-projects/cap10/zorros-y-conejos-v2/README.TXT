   Programación Orientada a Objetos con Java
   Una Introducción Práctica usando BlueJ
   Tercera Edición
   David J. Barnes and Michael Kolling
   Pearson Educación, 2007 

Un modelo de simulación predador-presa con zorros y conejos
dentro de un campo cerrado.

Esta es la segunda versión de la simulación. Esta versión
utiliza herencia para mejorar la estructura del código fuente.

Cómo comenzar:
    Crear un objecto Simulador.
    Luego, llame a:
        + simularUnPaso - para realizar un paso.
        + simular - debe proveer el número de pasos (ej. 10).
        + ejecutarSimulacionLarga - para simular 500 pasos.
